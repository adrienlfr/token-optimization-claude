## Common Pitfalls

❌ **Don't:**
- Compress prompts too aggressively (quality will suffer)
- Use Batch API for real-time systems
- Rely only on output constraints (format JSON, not just hope)
- Remove tools without testing

✅ **Do:**
- Start with one optimization at a time
- Measure before + after
- Test quality of outputs
- Iterate based on results
