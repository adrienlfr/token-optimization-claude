# Token Optimization for Claude

Recherche sur les leviers d'optimisation pour réduire la consommation de tokens avec Claude.

## Objectifs
- Comprendre le fonctionnement des tokens avec Claude
- Identifier les leviers d'optimisation
- Documenter les résultats et les meilleures pratiques

## Structure
- `/scripts` — Scripts de test et d'expérimentation
- `/results` — Résultats des expériences
- `/docs` — Documentation et notes de recherche
- `/data` — Données de test
